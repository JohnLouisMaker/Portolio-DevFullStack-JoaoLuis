# Stellar Showcase

Um portfólio moderno construído com React, TypeScript, Tailwind CSS e Vite.

## Tecnologias

- **React**
- **TypeScript**
- **Tailwind CSS**
- **Vite**
- **Framer Motion**
- **Lucide React**

## Como rodar o projeto

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

3. Para build de produção:
   ```bash
   npm run build
   ```
